# flutter_one_app

- 「ONE·一个」是由韩寒监制 [1]  ，原《独唱团》主创成员共同制作的一款文艺生活App ，官网地址：[「ONE·一个」](http://wufazhuce.com/)。此项目为高仿 ，仅做学习使用

-  此项目中的数据均来自one一个api真实有效数据

-  此项目持续更新中
